## Commit with a message
```bash
git commit -m "commit message"
```
or
```bash
git commit --message "commit message"
```

Creates a new commit with a message describing the
changes made. This command creates a new commit with
the changes you made to your local repository. The commit
message describes the changes made in this commit.
